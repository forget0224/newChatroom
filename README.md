# 即時聊天室

這是一個使用 Vue 3 和 Vite 開發的即時聊天室專案。

## 專案背景

此即時聊天室專案開發於 2023 年 5 月，主要技術棧包括 Vue.js 和 Tailwind CSS，並使用 Firebase database 和 Cloudflare 作為後端服務。

## 使用技術

- **前端**：Vue.js, Tailwind CSS
- **後端**：Firebase database, Cloudflare

## 功能

- 即時消息發送和接收
